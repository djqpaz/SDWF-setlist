# Sick Day with Ferris — Set List Builder

A drag-and-drop set list builder for the band.

## Setup

```bash
npm install
npm run dev
```

## Deploy to GitHub Pages

1. Create a GitHub repo named `sickday-setlist`
2. Push this project to it
3. Run:

```bash
npm run deploy
```

4. In GitHub repo settings → Pages → Source: `gh-pages` branch

Your app will be live at: `https://YOUR_GITHUB_USERNAME.github.io/sickday-setlist/`

## Features

- 48-song library with sort views (BPM, genre, year, popularity, emotional arc)
- Drag-and-drop set list builder
- Multiple shows with saved set lists
- Band members can save suggested sets under their name
- Print / export any set list
